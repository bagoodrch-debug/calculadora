Isto não é uma recomendação de investimento

O Distribui Rico é uma ferramenta educacional gratuita. Os percentuais e ativos exibidos aqui têm objetivo didático, não consideram seu perfil de investidor completo e não substituem a orientação de um profissional certificado.

Dúvidas frequentes

Perguntas frequentes

O Distribui Rico é uma recomendação de investimento?

Não. É uma ferramenta educacional que apresenta uma distribuição possível entre categorias de investimento, com base em percentuais que você mesmo pode ajustar. Nenhum conteúdo aqui deve ser interpretado como recomendação.

Os percentuais sugeridos são personalizados para o meu perfil?

A sugestão inicial é um ponto de partida educacional, não uma análise do seu perfil de investidor. Você pode — e deve — ajustar os percentuais para refletir seus próprios objetivos e tolerância a risco.

Meus dados ficam salvos em algum lugar?

A simulação roda inteiramente no seu navegador. Nenhum valor que você digitar é enviado a um servidor.

O Distribui Rico é realmente gratuito?

Sim, o simulador é 100% gratuito e não exige cadastro para ser usado.

Como os percentuais padrão são definidos?

A metodologia completa por trás dos percentuais sugeridos é explicada em detalhes na página de metodologia, incluindo as fontes e os critérios usados.
